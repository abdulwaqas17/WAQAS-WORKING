Operators in Java Script

  • Arthemetic Operators (+ ,- ,* ,/)
  • Unary Operators (a++/++a , a--/--a)
  • Assignment Operators (+=, -=, *=, /= ;| a += 1 _ a = a+1)
  • Comparison Operators (for two values, == equal to ..etc...)
  • Logical Operators
    1. Logical AND $$ [both must be true]
    2. Logical OR || [one must be true]
    3. Logical NOT ! [give opposite answer]
